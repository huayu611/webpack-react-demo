## 引用版本

* React 16.8.6
* Webpack 4.30.0
* Babel 7.4.3

## 下载
* `git clone https://github.com/huayu611/webpack-react-demo.git`
* `cd webpack-react-demo`

## 依赖下载
* `yarn install  / npm install`

##  开发配置 
* `yarn build  /  npm build`

##  构建配置 
* `yarn build  /  npm build`

##  介绍
*    Webpack和React集成。一个简单的demo。可以用这个Demo练习，也可以用来搭建初始框架。
